## Quick Start

1. **Initialize shadcn/ui**
   ```bash
   npx shadcn@latest init
   ```

2. **Install Framer Motion**
   ```bash
   npm install framer-motion
   ```

3. **Install a component**
   ```bash
   npx shadcn@latest add -r asth-ui animated-button
   ```
